#Sales Performance Dashboard Analysis using Ms Excel

##Project Objective
The objective of this project is to develop an interactive Sales Dashboard that enables stakeholders to monitor and analyze sales performance across different regions, product categories, and years. The dashboard provides key business metrics such as total sales, profit, quantity sold, and order count while allowing users to filter data by region, sub-category, and time period. The project aims to identify sales trends, evaluate profitability, compare product performance, and support data-driven decision-making to improve business growth and operational efficiency.
